Multiple Choice Practice Questions
1. What is the output of the following Python code?
print(3 * 'abc' + 'xyz')
abcxyz
abcabcxyzxyz
abcabcabcxyz
Error
Explanation: The * operator with strings performs repetition, so 'abc' repeated 3 times is 'abcabcabc', then concatenated with 'xyz'.
2. Which of these is NOT a valid Python variable name?
my_var
_private
3rd_var
var3
Explanation: Variable names cannot start with a number in Python.
3. What does the 'pass' keyword do in Python?
Terminates the program
Skips the current iteration in a loop
Acts as a placeholder for empty code blocks
Passes arguments to a function
Explanation: 'pass' is a null operation that does nothing, used when syntax requires a statement but no action is needed.
1. Which of these is NOT a primitive data type in Java?
int
char
String
boolean
Explanation: String is a class in Java, not a primitive type.
2. What is the default value of a boolean variable in Java?
true
false
null
0
Explanation: The default value for boolean instance variables is false.
3. Which keyword is used to inherit a class in Java?
implement
super
extends
inherit
Explanation: The 'extends' keyword is used to create a subclass that inherits from a superclass.
1. What will be logged to the console?
console.log(1 + "2" + "2");
5
122
"122"
32
Explanation: JavaScript performs type coercion. The + operator with a string performs string concatenation.
2. Which method creates a new array with all elements that pass a test?
map()
filter()
reduce()
forEach()
Explanation: The filter() method creates a new array with all elements that pass the test implemented by the provided function.
3. What is the purpose of the 'this' keyword in JavaScript?
Refers to the previous object
Refers to the current object
Refers to the parent object
Refers to the global object
Explanation: 'this' refers to the object that is executing the current function.
1. What does CSS stand for?
Computer Style Sheets
Creative Style System
Cascading Style Sheets
Colorful Style Sheets
Explanation: CSS stands for Cascading Style Sheets, which describes how HTML elements are displayed.
2. Which HTML5 element is used for playing video files?
<movie>
<media>
<video>
<play>
Explanation: The <video> element is used to embed video content in HTML5.
3. What is the purpose of the HTML 'alt' attribute?
To add a title to an image
To provide alternative text for an image
To align the image on the page
To specify the image source
Explanation: The 'alt' attribute provides alternative text for screen readers and when images can't be displayed.
